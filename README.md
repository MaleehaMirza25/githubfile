 Digital Diary

 About The Project
Digital Diary is a simple and secure web-based diary application built using HTML, CSS, and JavaScript.  
It allows users to write, save, edit, and delete personal notes digitally instead of maintaining a physical diary.

This project was created for personal learning and practice purposes.



Features
- Add new diary entries
- Edit existing entries
- Delete entries
- Save entries in browser (Local Storage)
- Clean and responsive user interface
- Simple and user-friendly design



 Built With
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage API



How To Run
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

No installation required.
